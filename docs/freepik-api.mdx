O Freepik oferece acesso via API ao modelo **Seedance**, o mesmo usado na plataforma web. É com ele que geramos os vídeos do personagem verde — 720p, 24fps, com proporção 4:5.

---

## Autenticação

Todas as requisições precisam de uma chave de API no header.

```bash
-H "x-freepik-api-key: SUA_CHAVE_AQUI"
```

Para obter sua chave: **freepik.com → conta → API Keys**.

---

## Gerar um vídeo

### Endpoint

```
POST https://api.freepik.com/v1/ai/video/seedance/text-to-video
```

### Headers

```http
Content-Type: application/json
x-freepik-api-key: SUA_CHAVE_AQUI
```

### Body

```json
{
  "prompt": "Seu prompt aqui",
  "duration": 5,
  "resolution": "720p",
  "ratio": "4:5"
}
```

### Parâmetros

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| `prompt` | string | sim | Descrição da cena em texto livre |
| `duration` | number | não | Duração em segundos — `5` ou `10` (padrão: `5`) |
| `resolution` | string | não | `"480p"`, `"720p"` ou `"1080p"` (padrão: `"720p"`) |
| `ratio` | string | não | `"16:9"`, `"9:16"`, `"4:5"`, `"1:1"`, `"21:9"` (padrão: `"16:9"`) |

---

## Fluxo da geração

A geração é **assíncrona**. Você envia o prompt, recebe um `task_id`, e fica consultando o status até o vídeo ficar pronto.

```
POST /text-to-video  →  task_id  →  GET /tasks/{id}  →  vídeo pronto
```

### 1. Enviar o prompt

```bash
curl -X POST "https://api.freepik.com/v1/ai/video/seedance/text-to-video" \
  -H "Content-Type: application/json" \
  -H "x-freepik-api-key: SUA_CHAVE_AQUI" \
  -d '{
    "prompt": "Pessoa vestida inteiramente com morphsuit verde vivo, cobrindo todo o corpo incluindo cabeça e rosto, de pé segurando um smartphone com calma. Quarto moderno de hotel de luxo, iluminação noturna azul cinematográfica. Profundidade de campo rasa, movimento lento de câmera, estética minimalista, visual ultra premium, 24fps.",
    "duration": 5,
    "resolution": "720p",
    "ratio": "4:5"
  }'
```

**Resposta:**

```json
{
  "data": {
    "task_id": "abc123-..."
  }
}
```

### 2. Verificar o status

```bash
curl "https://api.freepik.com/v1/ai/video/seedance/tasks/abc123-..." \
  -H "x-freepik-api-key: SUA_CHAVE_AQUI"
```

**Resposta enquanto processa:**

```json
{
  "data": {
    "task_id": "abc123-...",
    "status": "IN_PROGRESS"
  }
}
```

**Resposta quando pronto:**

```json
{
  "data": {
    "task_id": "abc123-...",
    "status": "COMPLETED",
    "generated": [
      {
        "url": "https://cdn.freepik.com/videos/abc123.mp4"
      }
    ]
  }
}
```

### Status possíveis

| Status | Significado |
|--------|-------------|
| `IN_PROGRESS` | Ainda gerando — continue consultando |
| `COMPLETED` | Pronto — `generated[0].url` tem o vídeo |
| `FAILED` | Falhou — tente novamente com o mesmo prompt |

## Exemplo completo em JavaScript

```javascript
const API_KEY = 'SUA_CHAVE_AQUI';
const BASE_URL = 'https://api.freepik.com/v1/ai/video/seedance';

async function gerarVideo(prompt) {
  // 1. Envia o prompt
  const res = await fetch(`${BASE_URL}/text-to-video`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-freepik-api-key': API_KEY,
    },
    body: JSON.stringify({
      prompt,
      duration: 5,
      resolution: '720p',
      ratio: '4:5',
    }),
  });

  const { data } = await res.json();
  const taskId = data.task_id;

  // 2. Polling até completar
  while (true) {
    await new Promise((r) => setTimeout(r, 5000)); // aguarda 5s

    const statusRes = await fetch(`${BASE_URL}/tasks/${taskId}`, {
      headers: { 'x-freepik-api-key': API_KEY },
    });

    const { data: task } = await statusRes.json();

    if (task.status === 'COMPLETED') {
      return task.generated[0].url;
    }

    if (task.status === 'FAILED') {
      throw new Error('Geração falhou');
    }
  }
}

// Uso
gerarVideo('Pessoa de morphsuit verde em quarto de hotel, segurando celular...')
  .then((url) => console.log('Vídeo:', url));
```

## Prompt padrão Zavvy

Cole direto no campo `prompt`. Já está ajustado para o modelo.

```
Pessoa vestida inteiramente com morphsuit verde vivo, cobrindo todo o corpo incluindo
cabeça e rosto, cor verde saturada (#00C840), sem feições visíveis, de pé segurando
um smartphone com calma. Movimentos lentos, suaves e confiantes.

Quarto moderno de hotel de luxo, janela panorâmica do piso ao teto com vista para
o oceano, cortinas de linho brancas, piso de madeira clara quente, cama plataforma
baixa com roupa de cama branca, homem de camiseta branca deitado por cima das cobertas.

Iluminação noturna azul cinematográfica. Profundidade de campo rasa, movimento lento
de câmera, estética minimalista, visual ultra premium, sombras suaves, look
cinematográfico 24fps.

Sem texto, sem letras, sem legendas, sem títulos sobrepostos — vídeo puramente visual.
```

Para os prompts completos de cada cena: [Prompts dos Vídeos](/docs/prompts)

## Configurações por formato

| Formato | `ratio` | `resolution` | `duration` |
|---------|---------|--------------|------------|
| Reels / Feed 4:5 | `"4:5"` | `"720p"` | `5` |
| Stories 9:16 | `"9:16"` | `"720p"` | `5` |
| Cinemascope 21:9 | `"21:9"` | `"720p"` | `10` |
| Quadrado 1:1 | `"1:1"` | `"720p"` | `5` |

## Dicas

<Callout type="tip">
**Polling ideal:** consulte o status a cada 5–10 segundos. Vídeos de 5s levam em média 60–90s para ficarem prontos.
</Callout>

<Callout type="warning">
**URL temporária:** o link do vídeo em `generated[0].url` expira. Faça o download imediatamente após receber o `COMPLETED`.
</Callout>

<Callout type="tip">
**Prompts em inglês performam melhor** no Seedance. Os prompts da Zavvy já foram validados — se for adaptar, mantenha os termos técnicos em inglês (morphsuit, bokeh, depth of field, cinematic lighting).
</Callout>
